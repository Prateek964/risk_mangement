## Arithmetic Operations

Python achieves arithmetic operations using below operators:

| Opearation         | Operator | Example | Result |
| :----------------- | :------- | ------- | ------ |
| Addition           | +        | 7+5     | 12     |
| Subtraction        | -        | 7-5     | 2      |
| Multiplication     | X        | 7 X 5   | 35     |
| Division           | /        | 7/5     | 1      |
| Modulo (Remainder) | %        | 7%5     | 2      |

</div>
